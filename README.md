# YouTube Download

A Claude Code skill for downloading videos and audio from YouTube using yt-dlp.

## Documentation

See [SKILL.md](./SKILL.md) for complete documentation and usage instructions.

## Installation

To use this skill with Claude Code, clone it into your skills directory:

```bash
git clone https://github.com/HartreeWorks/skill--youtube-download.git ~/.claude/skills/youtube-download
```

Then restart Claude Code to load the skill.
